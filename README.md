# Nach Dakwale's Portfolio

A personal portfolio website showcasing my projects and experience in AI/ML, developer products, payments, cryptocurrency, and trading technologies.

## Features

- Clean, modern design
- Responsive layout
- Project showcase with expandable details
- Interactive photo gallery
- Contact information and social links

## Structure

```
portfolio/
├── index.html
├── projects.html
├── styles/
│   └── main.css
├── images/
└── js/
    └── main.js
```

## Setup

1. Clone the repository:
```bash
git clone https://github.com/yourusername/portfolio.git
```

2. Open `index.html` in your browser to view the website locally.

## Deploy to GitHub Pages

1. Push this repo to GitHub.
2. For a user site: Rename the repo to `username.github.io` (replace `username` with your GitHub username).
3. Go to **Settings → Pages** and enable GitHub Pages (Source: main branch).
4. Your site will be live at `https://username.github.io`.
