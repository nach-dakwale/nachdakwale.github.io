# Nach Dakwale's Portfolio

A personal portfolio website showcasing my projects and experience in AI/ML, developer products, payments, cryptocurrency, and trading technologies.

## Features

- Clean, modern design
- Responsive layout
- Project showcase
- About me section
- Contact information

## Structure

```
portfolio/
├── index.html
├── about.html
├── projects.html
├── styles/
│   └── main.css
└── images/
```

## Setup

1. Clone the repository:
```bash
git clone https://github.com/yourusername/portfolio.git
```

2. Open `index.html` in your browser to view the website locally.

## Technologies Used

- HTML5
- CSS3
- Responsive Design
- Modern CSS Features (Grid, Flexbox)
